

# Kanban Board with AI Chatbot

A gorgeous, interactive Kanban board inspired by the modern dark-themed project management UI from your reference images.

## Design Style
- **Dark theme** with rich gradients and glassmorphism effects matching your reference images
- Smooth drag-and-drop interactions with subtle animations
- Clean typography, rounded cards with colored priority indicators and category tags
- Sidebar navigation with a sleek, modern feel

## Kanban Board
- **Three columns**: To-Do, In Progress, and Done
- **Task cards** with: title, description, due date, priority level (color-coded), and category tags
- **Drag and drop** to move tasks between columns
- **Add, edit, and delete** tasks with a clean modal form
- **Search and filter** tasks by priority or category

- **Main Board** page (protected route) with the Kanban view
